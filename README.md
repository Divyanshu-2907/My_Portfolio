# Divyanshu Kumar - Developer Portfolio 🚀

A modern, responsive developer portfolio designed to showcase my projects, professional experience, and skills across **Full-Stack Web Development**, **AI Systems**, and **Embedded Robotics**.

## ✨ Features

- **Modern UI/UX:** Built with a clean dark theme, glassmorphism elements, and smooth scroll animations.
- **Custom Interactions:** Features a custom cursor and dynamic hover effects for an engaging user experience.
- **Live GitHub Stats:** Embeds a real-time GitHub contribution heatmap.
- **Fully Functional Contact Form:** Integrated with **EmailJS** for a seamless, non-redirecting form submission that instantly notifies me and sends an automated professional receipt to the sender.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** Vanilla CSS3 (Custom properties, Flexbox/Grid, Animations, Media Queries)
- **Logic:** Vanilla JavaScript (ES6+)
- **Integrations:** EmailJS (Form handling)
